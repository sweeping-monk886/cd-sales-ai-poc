# CD销售AI助手 POC

AI赋能CD领域销售和经销商，提升效能并提供销售洞察和行动建议。

## 功能特性

### 📚 问知能力
- 快速检索产品知识、销售话术、FAQ
- 支持多产品、多品类知识库
- 智能匹配相关文档

### 📊 问数能力
- 自然语言查询销售数据
- 自动生成SQL并执行
- 数据可视化展示
- 智能数据解读

### 💡 洞察能力
- KPI完成率预警
- 销售趋势分析
- 区域对比洞察
- 主动行动建议

## 快速开始

### 环境要求
- Python 3.8+
- pip

### 安装步骤

1. 克隆项目
```bash
cd cd-sales-ai-poc
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 初始化数据库
```bash
python data/init_db.py
```

4. 启动应用
```bash
streamlit run app.py
```

5. 访问应用
打开浏览器访问 http://localhost:8501

## 项目结构

```
cd-sales-ai-poc/
├── app.py                 # 主应用
├── requirements.txt       # 依赖
├── README.md             # 说明文档
├── knowledge_base/       # 知识库文档
│   ├── 产品A_智能冰箱ProX1.md
│   ├── 产品B_智能空调StarV3.md
│   └── 销售FAQ.md
├── data/
│   ├── init_db.py        # 数据库初始化
│   └── sales.db          # SQLite数据库
└── utils/
    ├── agent.py          # Agent核心模块
    ├── knowledge.py      # 问知模块
    ├── data_query.py     # 问数模块
    └── insight.py        # 洞察模块
```

## 数据说明

本POC使用模拟数据，包含：
- 20家经销商（华东/华南/华中/华北4个区域）
- 7款产品（冰箱/空调/洗衣机3个品类）
- 6个月KPI数据
- 约600条销售记录

## 技术栈

- **前端**: Streamlit
- **后端**: Python
- **数据库**: SQLite
- **AI框架**: 自研Agent + 规则引擎
- **可视化**: Plotly

## 扩展方向

1. **接入真实LLM**: 集成GPT-4/通义千问等提升问答质量
2. **向量知识库**: 使用FAISS/Milvus提升检索精度
3. **Text-to-SQL**: 使用LLM替代规则生成SQL
4. **实时数据**: 对接企业真实数据库
5. **多轮对话**: 支持上下文理解
6. **权限控制**: 不同角色看到不同数据

## 注意事项

- 本POC为演示版本，使用模拟数据
- 生产环境需要：
  - 接入真实业务数据
  - 添加用户认证
  - 优化性能和安全性
  - 添加日志和监控
